## üöÄ The Problem
Campus finance is currently **fragmented, slow, and opaque**. 
- Students struggle with splitting bills for group projects or outings.
- Club fundraising is chaotic, with no transparency on where funds go.
- Event ticketing is prone to scalping and fake tickets.
- P2P payments often involve friction and delays.

## üí° The Solution: VIT Algo Pay
**VIT Algo Pay** is a comprehensive, **Web3-native financial ecosystem** built on the **Algorand Blockchain**. It unifies payments, bill splitting, fundraising, and ticketing into a single, futuristic SuperApp designed for the digital-native student.

### ‚ú® Key Features

#### 1. ‚ö° Instant P2P Transfers
- **Zero Latency:** Send ALGO or ASAs (Campus Tokens) instantly.
- **QR Scanning:** built-in camera integration for seamless in-person payments.
- **Low Fees:** < 0.001 ALGO per transaction.

#### 2. ü§ù Atomic Bill Split
- **Trustless Splitting:** Create atomic group transactions for shared expenses.
- **No Middleman:** Funds move directly between peers; no "holding" wallet.
- **Transparent:** Everyone verifies their share before signing.

#### 3. üöÄ DAO Fundraising
- **Transparent Crowdfunding:** Clubs can raise funds with public accountability.
- **Smart Contracts:** Logic ensures funds are only used as intended (simulated logic).
- **Real-time Progress:** Visual tracking of campaign goals.

#### 4. üéüÔ∏è NFT Ticketing
- **Verifiable Access:** Tickets are minted as Non-Fungible Tokens (ASAs).
- **Anti-Scalping:** Ownership is traceable on-chain.
- **Holographic UI:** Premium experience for event organizers and attendees.

## üì¶ Installation & Setup

### Prerequisites
- Node.js (v16+)
- **Pera Algo Wallet** (Mobile App) -> Set to **TestNet**.

### 1. Clone & Install
```bash
git clone https://github.com/your-username/campuspay.git
cd campuspay/frontend
npm install
```

### 2. Run Locally
```bash
npm run dev
```
Open `http://localhost:5173` in your browser.

### 3. Connect & Play
- Click **"CONNECT WALLET"** and scan with Pera App.
- Use the **TestNet Dispenser** to get free ALGO.
- Try the **"Instant Transfer"** or **"Bill Split"** modules!
